# 登录页面

用户登录页面，提供邮箱密码登录功能。

## 登录页面

URL: /login

### 页面元素

- 邮箱输入框 (input, placeholder="请输入邮箱")
- 密码输入框 (input, type=password, placeholder="请输入密码")
- 记住我复选框 (checkbox)
- 登录按钮 (button, text="登录")
- 忘记密码链接 (link, text="忘记密码？")
- 注册链接 (link, text="没有账号？去注册")

### 用户操作

- 输入邮箱地址
- 输入密码
- 可选勾选"记住我"
- 点击登录按钮
- 点击忘记密码跳转到重置密码页
- 点击注册链接跳转到注册页

### 预期结果

- 登录成功后跳转到 /dashboard
- 邮箱为空时显示"请输入邮箱"校验提示
- 密码为空时显示"请输入密码"校验提示
- 邮箱格式不正确显示"邮箱格式不正确"
- 密码错误显示"邮箱或密码错误"
- 勾选记住我后关闭浏览器重新打开应自动登录
- 连续输错密码5次后显示"账号已锁定，请30分钟后重试"

## 注册页面

URL: /register

### 页面元素

- 用户名输入框 (input, placeholder="请输入用户名")
- 邮箱输入框 (input, placeholder="请输入邮箱")
- 密码输入框 (input, type=password)
- 确认密码输入框 (input, type=password)
- 注册按钮 (button, text="注册")
- 返回登录链接 (link, text="已有账号？去登录")

### 用户操作

- 填写用户名、邮箱、密码、确认密码
- 点击注册按钮
- 点击返回登录链接

### 预期结果

- 注册成功后跳转到 /login 并显示"注册成功，请登录"
- 用户名少于2个字符显示"用户名至少2个字符"
- 邮箱已被注册显示"该邮箱已被注册"
- 两次密码不一致显示"两次密码不一致"
- 密码少于8位显示"密码至少8位"
