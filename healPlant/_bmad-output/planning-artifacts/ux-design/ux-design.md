# UX Design Specification: HealPlant

**Author:** Dreamer
**Date:** 2026-05-22

---

## 1. 设计概览

### 1.1 产品定位
HealPlant 是一个双面架构的绿植综合经营网站：前台面向公众展示、后台面向管理员私密管理。UX 设计需同时服务两类截然不同的使用场景。

### 1.2 设计目标
- 前台：清新治愈、温馨亲切，让访客感受到专业且有温度的绿植生活方式
- 后台：高效实用、数据清晰，让零技术背景的管理员能快速完成日常操作
- 整体：响应式布局，移动优先，手机和桌面体验一致

## 3. 核心体验定义

### 3.1 前台访客体验
- **第一印象（0-3秒）：** 看到草绿色背景上的清新植物图片和温暖的欢迎语，感受到"这是一个热爱植物的人"
- **探索体验（3-30秒）：** 通过清晰的导航快速找到感兴趣的内容区域，内容丰富但不杂乱
- **互动体验：** 可以留言、预约，有被回应的感觉

### 3.2 后台管理员体验
- **效率优先：** 最常用操作（录入进货、记录销售）不超过 3 次点击
- **数据可见：** 关键指标（今日利润、库存、待处理预约）在首页一目了然
- **容错友好：** 所有删除操作需确认，误操作可撤回

## 4. 情感化设计

### 4.1 前台情感调性
- **温暖治愈：** 圆角卡片、柔和渐变、植物纹理背景
- **趣味活泼：** 趣味广场使用活泼的卡片动效，内容呈现形式多样
- **专业信赖：** 服务页和产品页使用规整布局，案例对比图增加信任感

### 4.2 后台情感调性
- **沉稳高效：** 白色基底、清晰的表格和数据卡片
- **成就激励：** 报表页面使用积极的数据可视化，盈利数据突出显示

## 8. 交互模式

### 8.1 表单交互
- 实时验证：输入框失焦后立即验证
- 错误提示：红色边框 + 下方错误文字
- 成功反馈：绿色 Toast 消息 2 秒后自动消失
- 保存：显式保存按钮，不自动保存

### 8.2 列表交互
- 新增：页面右上角主按钮
- 编辑：行内编辑按钮或点击行进入详情
- 删除：红色按钮 + 二次确认弹窗
- 筛选：顶部筛选栏，支持多条件组合
- 搜索：实时搜索，300ms 防抖

### 8.3 数据录入快捷流
- 进货录入：品种可从品种库下拉选择或快速新增
- 销售录入：品种可从库存中选择，自动带入进价
- 定价切换：Radio 按钮切换模式，输入框动态变化

### 8.4 视频交互
- 上传：拖拽或点击上传，显示进度条
- 播放：点击缩略图进入播放，支持全屏
- 列表：网格视图，悬停显示时长和操作按钮

## 9. 响应式策略

### 9.1 断点定义

| 名称 | 范围 | 布局 |
|------|------|------|
| mobile | < 640px | 单列 |
| tablet | 640px - 1024px | 双列 |
| desktop | > 1024px | 多列 + 侧边栏 |

### 9.2 关键适配规则

**前台：**
- 导航：桌面水平菜单 → 移动汉堡菜单
- 卡片：桌面 3 列 → 平板 2 列 → 手机 1 列
- 图片：桌面并排 → 手机堆叠
- 视频：全宽自适应，16:9 比例

**后台：**
- 导航：桌面侧边栏固定 → 移动抽屉菜单
- 表格：桌面完整表格 → 移动卡片列表
- 表单：桌面双列 → 移动单列
- 图表：桌面并排 → 移动纵向堆叠

### 9.3 可访问性
- 所有交互元素 ≥ 44px 触摸区域
- 色彩对比度 ≥ 4.5:1 (WCAG AA)
- 图片配 alt 文字
- 表单配 label 关联
- 键盘可导航

### 10.1 前台导航
- **桌面：** 顶部固定导航栏，Logo 左对齐，菜单项水平排列
- **移动：** 顶部 Logo + 汉堡图标，点击弹出全屏菜单
- **菜单项：** 首页、个人简介、趣味广场、养护服务、产品展示、养花科普、互动留言

### 10.2 后台导航
- **桌面：** 左侧固定侧边栏，240px 宽，分组折叠
- **移动：** 底部 Tab 栏（首页/内容/经营/视频/更多）+ 抽屉菜单展开完整导航
- **分组：** 内容管理、经营管理、素材管理、学习/预约

## 11. 页面流转

### 11.1 访客预约流程
```
首页 → 养护服务页 → 浏览服务项目 → 查看案例 → 填写预约表单 → 提交成功提示
```

### 11.2 管理员日常操作流程
```
登录 → Dashboard(查看今日概况)
  → 进货录入(进了新货) → 定价(给新品定价)
  → 销售录入(收摊记账) → 查看报表(看今日盈亏)
  → 发布内容(拍了新图/视频) → 登出
```

### 11.3 管理员内容发布流程
```
后台首页 → 趣味内容管理 → 新增内容 → 选择分类 → 编辑图文/上传视频 → 设置公开 → 发布 → 前台可见
```
