# Sprint 2 — 品种资料库 + 智能定价 + 文件上传

- AC-3: 品种列表支持按热度、难度筛选
- AC-4: 前台展示开关可切换，可编辑独立的前台公开描述
- AC-5: 封面图上传（依赖 S-6.1）
- AC-6: 对应 PRD FR-26

### 技术任务
1. 创建品种详情页 `/admin/varieties/[id]/page.tsx`
2. 创建编辑表单组件，支持全字段编辑
3. 品种列表增加热度、难度筛选下拉
4. 前台展示开关 + 公开描述编辑区域
5. 更新 `PUT /api/admin/varieties/:id` 确保全字段更新正确

## Story S-3.2: 双模式智能定价

**作为** 管理员，**我可以** 对品种使用手动售价或成本涨幅比例两种方式定价，**以便** 灵活设定售卖价格。

### 验收标准
- AC-1: 品种详情页包含定价面板，Radio 切换手动/涨幅两种模式
- AC-2: 手动模式：输入售价 → 保存成功
- AC-3: 涨幅模式：输入百分比 → 系统自动计算售价（= 进价 × (1 + 百分比/100)）→ 显示计算结果 → 保存
- AC-4: 涨幅模式下如该品种无进货记录，提示用户先录入进货
- AC-5: 定价历史列表展示（时间、模式、参数、结果售价）
- AC-6: 对应 PRD FR-27, FR-28, FR-29, FR-30

### 技术任务
1. 创建 PricingPanel 组件
2. 集成到品种详情页
3. 定价历史列表组件
4. API 已就绪（`POST/GET /api/admin/varieties/:id/pricing`）

## Story S-6.1: 文件上传服务

**作为** 系统，**我需要** 提供图片上传 API，**以便** 支撑品种封面图、内容图片等上传需求。

### 验收标准
- AC-1: `POST /api/admin/upload/image` 接受图片文件上传
- AC-2: 类型白名单：jpg, jpeg, png, webp, gif
- AC-3: 大小限制 10MB
- AC-4: 文件以随机名存储到 `/uploads/images/` 目录
- AC-5: 返回可访问的图片 URL
- AC-6: 对应 PRD FR-45

### 技术任务
1. 创建上传 API Route Handler（multipart/form-data 解析）
2. Next.js 静态文件配置，使 `/uploads/` 可访问
3. 文件名随机化（UUID）
